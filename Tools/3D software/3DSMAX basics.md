## Operations

View:
- scroll: move
- scroll + alt: rotate
- scroll up and down: zoom

Select:
- click
- box selection (window/crossing)

## Import CAD to 3dsmax

In CAD: 

- select --> W(WBLOCK) --> save as a new file

In 3dsmax: 

- cuctomiz --> units setup --> mm
- file --> import
- select all layout --> group
